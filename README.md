## ✨ すぐに開く（推奨2通り）

### 方法1: ダブルクリック（最速・依存ゼロ）

`index.html` を **Chrome / Edge / Safari / Firefox** などモダンブラウザで直接開いてください。
用語集・クイズのデータはJSにインライン化してあるので、`file://` でも全機能動きます。

### 方法2: ローカルサーバ（パスエラー等が出た場合の保険）

Python 3 が入っていれば、フォルダ直下で:

```bash
python -m http.server 8000
```

PowerShellの場合:

```powershell
python -m http.server 8000
```

ブラウザで `http://localhost:8000/` を開く。

## 🛠 カスタマイズの入口

| やりたいこと | 編集ファイル |
|---|---|
| サイドバーの章を増減 | `assets/js/sidebar.js` の `NAV` 配列 |
| 色・余白・フォント変更 | `assets/css/base.css` の `:root` |
| 用語集に語を追加 | `assets/js/glossary.js` の `DATA.categories` |
| クイズ問題を追加 | `assets/js/quiz.js` の `QUIZ` 配列 |
| 章本文の修正 | `pages/chapter*.html` を直接編集 |

## 🔍 トラブルシューティング

| 症状 | 対処 |
|---|---|
| サイドバーが出ない | ブラウザのJSが無効になっていないか確認 |
| フォントが読み込まれない | オフライン環境ならGoogle FontsのCDNが失敗します。`<link href="...fonts...">` を削れば既定フォントで表示 |
| メモが消えた | ブラウザのプライベートウィンドウだとセッション終了で消えます。通常ウィンドウで使ってください |
| クイズが進まない | localStorage の `quiz:result:v1` を消すか、ページ末「🔄 もう一度」を押す |
